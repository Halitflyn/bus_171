# 🚍 Розклад маршрутки №171 (Басівка ↔ Львів)

Веб-додаток для зручного перегляду актуального розкладу маршруту №171. Створений з акцентом на швидкість, зручність використання на мобільних пристроях та роботу в умовах нестабільного інтернету.

## ✨ Основні можливості

* **Розумне підсвічування:** Сайт автоматично знаходить найближчий рейс, показує час до відправлення та приховує рейси, що вже поїхали.
* **Автоматичний вибір дня:** Додаток сам визначає, який сьогодні день (будні чи вихідні), і відкриває відповідну вкладку.
* **PWA (Progressive Web App):** Сайт можна встановити на головний екран смартфона як звичайний додаток. Він працює **офлайн** завдяки Service Worker.
* **Темна тема:** Автоматично адаптується під налаштування системи або перемикається вручну.
* **Динамічні ціни:** Актуальна вартість проїзду по всіх населених пунктах (Сокільники, Малечковичі, Наварія, Годовиця, Басівка).

## 🛠 Технології

* **HTML5 / CSS3:** Сучасна верстка з використанням Flexbox, Grid та CSS Variables.
* **JavaScript (Vanilla JS):** Логіка підсвічування рейсів, робота з табами та темою без сторонніх бібліотек.
* **JSON:** Всі дані про рейси та ціни винесені в окремий файл `schedule.json` для зручного оновлення.
* **Service Workers:** Забезпечують кешування та офлайн-доступ.
* **GitHub Actions:** Налаштовано автоматичний деплой через `static.yml`.

## 📂 Структура проєкту

* `index.html` — головна сторінка та структура сайту.
* `style.css` — стилі, адаптивний дизайн та теми.
* `script.js` — логіка обробки часу, підсвічування та ініціалізація додатку.
* `schedule.json` — "серце" сайту з даними розкладу та цін.
* `service-worker.js` — магія офлайн-доступу.
* `manifest.json` — налаштування для встановлення сайту як PWA.

## 🚀 Як оновити розклад

Щоб змінити час відправлення або ціни, достатньо відредагувати файл `schedule.json`. Після комміту зміни автоматично з'являться на сайті завдяки GitHub Actions. Не забудь змінити поле `"lastUpdated"`, щоб користувачі бачили дату актуальності.
